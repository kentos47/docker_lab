# Skyrim TBRPG — Console RPG in C++
##### Was made by Nikita Shubin


This project is a text-based role-playing game in the style of The Elder Scrolls, written in C++. The game runs in the terminal and invites you to immerse yourself in the world of Skyrim with quests, monsters, and inventory.

## Running with Docker

For easy execution, the application is packaged in a Docker container.
[You only need Docker installed.](https://docs.docker.com/engine/install/)
## Building the Image

- Clone the repository and navigate to its root folder
```bash
git clone https://github.com/ваш-username/Skyrim-TBRPG-Cpp-Project
cd Skyrim-TBRPG-Cpp-Project
```

- Build the Docker image
```bash
docker build -t skyrim .
```

- Run the container
```bash
docker run -it skyrim
```
